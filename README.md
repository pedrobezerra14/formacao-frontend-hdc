# Formação Front-End - Hora de Codar 🚀

Este repositório é o meu "diário de bordo" durante a **Formação Front-End** do Matheus Battisti. Aqui registro toda a minha evolução, desde os primeiros seletores CSS até aplicações complexas com React e TypeScript.

---

## 📌 Sobre o Curso

- **Fundamentos:** HTML5 Semântico e CSS3 Avançado (Flexbox/Grid).
- **Lógica de Programação:** JavaScript ES6+ e manipulação ativa do DOM.
- **Ecossistema React:** Hooks, Context API, React Router e State Management.
- **Tipagem com TypeScript:** Desenvolvimento escalável e seguro.

---

## 📂 Organização do Repositório

A estrutura foi pensada para separar a teoria da prática:

* **`01-FUNDAMENTOS/`**: Exercícios de fixação e conceitos teóricos.
    * `HTML/`: Tags, semântica e formulários.
    * `CSS/`: Estilização, layouts responsivos e animações.
    * `JS/`: Lógica, arrays, objetos e funções.
* **`02-PROJETOS/`**: Aplicações reais construídas durante o curso.
    * `project_imc_js`: Calculadora de IMC com manipulação de DOM.
    * `project_todoList`: Gerenciador de tarefas com persistência.
    * `project_iphone`: Estilização avançada e interatividade.
    * *(E outros projetos em desenvolvimento...)*
